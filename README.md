# GamemodeCMDs #

GamemodeCMDs is a plugin for pocketmine wich add short commands to switch your gamemode! | GamemodeCMDs ist ein Plugin für Pocketmine welches kurze Befehle zum ändern deines Gamemodes addet!

# Commands #

- /gm0
- /gm1
- /gm2
- /gm3

# Permissions #

- gm0.cmd -> /gm0
- gm1.cmd -> /gm1
- gm2.cmd -> /gm2
- gm3.cmd -> /gm3

# Other Languages #

Just change the messages in config.yml :D
